# El Badeil for Recycling

A website for replacing used cooking oil with free cleaning products.

## Features
* Pure Code
* 3 Sliders
* 2 Languages
* Number Counter
* Contact Form Linked with Firebase
* Animations
* Elegant Design
* Fast Loading
* Cross-Browser
* Fully Responsive
* Articles Pages
* Much More
